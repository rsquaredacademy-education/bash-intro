Chapter 2 Introduction to Shell

2.1 Introduction

In this chapter, we will learn the basics of using command line. We will learn just enough topics to be able to use Git & GitHub. If you want to learn more, we have listed a few resources at the end of the chapter for reference.

2.2 Directory

2.2.1 Current directory

To get the current directory, use pwd (present working directory).

pwd
## /cygdrive/j/R/ebooks/bash-intro

2.2.2 Navigate

To navigate to a sub directory, use cd followed by the name of the directory. You donâ€™t need to type the full name of the directory, type a few letters and use TAB for auto completion. In the below example, we navigate to the docs directory using cd.

cd docs
pwd
## /cygdrive/j/R/ebooks/bash-intro/docs

To return to the main directory, use cd ... Remember there is a single space between cd and the 2 dots.

cd ..
pwd
## /cygdrive/j/R/ebooks

2.2.3 View contents

To list all the sub folders and files in a directory, use ls. If you want to list hidden folders and files, use ls -a.

2.3.5 Move files

To move files to a different folder or sub folder, use mv. In the below example, we use mv to move home.Rmd to the examples folder.

mkdir examples
cp index.Rmd home.Rmd
mv home.Rmd examples
cd examples
ls
## home.Rmd

2.4 Word count

wc will return the number of characters, words and lines in a file.

# characters, words and lines
wc style.css
##  14  26 172 style.css

The following can be used to return only one of the above:

  • -c for characters
  • -w for words
  • -l for lines

2.4.0.1 Characters

# characters
wc -c style.css
## 172 style.css

2.4.0.2 Words

# words
wc -w style.css 
## 26 style.css

2.4.0.3 Lines

# lines
wc -l style.css 
## 14 style.css
